                Call Bind and Apply are different ways to invoke a function. The first parameter is always 
                an obj.

                In Call the function methood is borrowed & is passed in form of Argument,
                In Apply the function methood is borrowed & is passed in form of Array,
                whereas In Bind borrows the function method and passes the object as an argument
                and returns a new function

                let obj={
                    name:"Mike",
                    age:25
                }
                
                
                function print(salary)

                    console.log(${this.name} is getting ${salary});

                print.call(obj,$50000);
